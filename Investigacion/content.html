Investigacion de conceptos de programacion web

¿Que es un form?

Un formulario en html es una seccion de contenido normal, codigo y elemtos especiales llamados "controles" (radio button, button, textbox, etc.) con los cuales los usuarios "completan" el formulario con la informacion que sea requerida

¿Como crear un Form?

<FORM action="http://algunsitio.com/prog/usuarionuevo" method="post">
<P>

    <INPUT type="text" id="nombre"><BR>

Controles de texto

Para que el usuario pueda ingresar datos de tipo texto generalmente se utlizan inputs de tipo texto, estas tiene dos principales variantes, texto de una sola linea y texto de multiples lineas, que su uso varia dependiendo que se requiera que el usuario ingrese dentro de el objeto

Cajas de texto de una linea

Las imput box de texto comunes solo adminten que el usuario solo pueda escribir una linea de texto, en el siguiente codigo de ejempo esta el codigo para insertar un imput box de una linea

    Ejemplo de creacion de un imput text box de una linea

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title%gt;
</head>
<body>
    <input type="text">
</body>
</html>

Cajas de texto de multiples lineas

En caso de ser requerido hay una etiqueta que permite el ingreso de varias lineas de texto, se utliza la etiqueta <textarea cols="30" rows="10"></textarea> en la cual se tendra que especificar el numero de filas ("rows") y columnas ("cols") que determinan el tamaño del objeto

        Ejemplo de creacion de un cuadro de texto largo

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title%gt;
</head>
<body>
    <textarea cols="30" rows="10"></textarea>
</body>
</html>

Controles de verificacion

En ciertos casos es requerido que el usuario solo pueda seleccionar entre un conjunto de opciones pre establecidad, para eso se utlizan etiquetas tales como checkbox ,radiobutton o listas desplegables

checkbox

las checkbox permiten crear varias opciones de seleccion, de las cuales los usuarios pueden seleccionar una o varias segun demande la situacion

        Ejemplo de creacion checkboxs

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title%gt;
</head>
<body>
    <input type="checkbox" id="checkbox1"><label>Opcion 1</label>
    <input type="checkbox" id="checkbox2"><label>Opcion 2</label>
    <input type="checkbox" id="checkbox3"><label>Opcion 3</label>
</body>
</html>

Radiobutton

Los radiobutton tienen un funcionamiento similar a los checkbox con la diferencia que en este caso el usario solo prodra seleccionar solamente una opccion, para que esta funcion sea valida en el atributo name del radiobutton hay que asignarle el mismo valor al conjunto de radiobuttons que queramos que se le aplique

        Ejemplo de creacion radiobutton

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <input name="radio" type="radio" id="radio1"><label>Opcion 1</label>
    <input name="radio" type="radio" id="radio2"><label>Opcion 2</label>
    <input name="radio" type="radio" id="radio3"><label>Opcion 3</label>
</body>
</html>

Listas

La etiqueta select tiene la funcion de crea una lista desplegable de la cual se podra elegir una de las opciones que esta contenga, normalmente el contenido de la etiqueta se tiene que llenar a mano, pero mediante metodos de un script se pueden llenar el contenido de manera automatica

        Ejemplo de creacion de una lista

<!DOCTYPE html>
<html lang="en">
    <head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <select id="lista">
        <option>Opcion 1</option>
        <option>Opcion 2</option>
        <option>Opcion 3</option>
    </select>
</body>
</html>

Botones

Uno de las principales etiquetas de html son los botones a los cuales se le pueden asignar un sin fin de funcionalidades tanto de llamar a funciones de un script o dirigit a una parte de la pagina, en el caso especifico de un formulario los dos principales botones empleados son botones de envio y botones de reinicio

Boton submit

El botón submit sirve para enviar el formulario una vez el usuario ha rellenado todos sus campos y pulsa en este botón. El texto por defecto del botón será Enviar (o equivalente según el idioma). Podemos modificar el texto del botón mediante el atributo value.

        Ejemplo de boton submit

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title%gt;
</head>
<body>
    <form name="formulario" method="post" action="send.php">
        Usuario: <input type="text" name="usuario" />
        <input type="submit" value="Enviar formulario" />
    </form>
</body>
</html>

Boton reinicio

El botón reset nos ofrece la posibilidad de disponer de un botón que al pulsarse borre completamente el formulario y reestablece todos sus campos de entrada de datos a sus valores por defecto iniciales. Es muy útil si queremos que el usuario disponga de la posibilidad de borrar todos los campos del formulario fácilmente y no tener que hacerlo uno por uno.

        Ejemplo de boton reinicio

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title%gt;
</head>
<body>
    <form name="formulario" method="post">
        Usuario: <input type="text" name="usuario" />
        <input type="submit" value="Enviar formulario" />
        <input type="reset" value="Borrar formulario" />
    </form>
</body>
</html>
